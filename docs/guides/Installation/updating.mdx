# Updating

Auto-update is only available on cargo at the moment.
Please uninstall and reinstall dora if you wish to update dora with github release,
but don't forget to update python package as well with:

```bash
pip install dora-rs --upgrade
```

## Cargo and Pip

```bash
cargo install dora-cli
cargo install dora-coordinator
cargo install dora-daemon
pip install dora-rs --upgrade
```

## Source

```bash
## Go to dora local repository
cd dora
cargo build --release --all

## Python API
cd apis/python/node
maturin develop --release
cd ../../..
```
