# Dora Drives

[Dora drives](https://github.com/dora-rs/dora-drives) is a starter kit project to use dora-rs for autonomous driving.
In order to do that we build a dataflow splitting autonomous driving into several nodes from perception to control.
We can then feed the dataflow with data from a simulated car within the Carla Simulator.

The goal of dora drives is to be a simple and instructive approach into autonomous driving.

It does this by implemnating a set of operators:
